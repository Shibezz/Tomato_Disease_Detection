🍅 Tomato Disease Detection using VGG19

A deep learning-based project leveraging VGG19 for classifying tomato plant leaf diseases with high accuracy.
🔍 Overview

This project applies transfer learning using the VGG19 architecture, pretrained on ImageNet, to detect and classify common tomato plant diseases. It helps in early diagnosis and can serve as a tool to assist farmers or agricultural experts.
🧠 Model Highlights

    📚 Architecture: VGG19 (Transfer Learning)

    🏷️ Classification: Multiclass image classification of tomato leaf diseases

    🏋️ Trained On: Custom dataset of tomato plant images

    💾 Output: model.h5 (tracked with Git LFS)
